agregando la configuracion de todo el dotfile:

primero inciamos con actualizar el sistema

sudo pacman -Syyu

ahora isntalamos yay y aceptamos todos los permisos:

sudo pacman -S --needed git base-devel && git clone https://aur.archlinux.org/yay.git && cd yay && makepkg -si

luego instlamos los nesesario primero:

sudo pacman -S hyprland

ahora instalamos los drivers de intel:

sudo pacman -S mesa lib32-mesa
sudo pacman -S vulkan-intel lib32-vulkan-intel vulkan-icd-loader lib32-vulkan-icd-loader
sudo pacman -S intel-media-driver libva-utils
sudo pacman -S mesa-demos vulkan-tools
sudo pacman -S intel-gpu-tools

quitamos si es nesesario xf86-video-intel:

sudo pacman -Rns xf86-video-intel

ahora vamos con nvidia:

sudo pacman -Syu
sudo pacman -S nvidia nvidia-utils lib32-nvidia-utils nvidia-settings nvidia-prime

Desactivamos nouveau (para que no choque con nvidia)

Creamos el archivo:

sudo nano /etc/modprobe.d/blacklist-nouveau.conf
Con este contenido:

blacklist nouveau
options nouveau modeset=0

luego Añadimos parametros al kernel (systemd-boot) en mi caso

Con systemd-boot, los parametros se ponen en los archivos de /boot/loader/entries/.

ls /boot/loader/entries

Veras algo como arch.conf o arch-linux.conf o algo similar.

Busca la línea que empieza con options.
Ahí estarán cosas como root=... rw quiet splash etc.

Añade al final de esa línea estos parametros:

rd.blacklist=nouveau modprobe.blacklist=nouveau nvidia_drm.modeset=1

ahora vamos a cargar los módulos de NVIDIA (e Intel) muy temprano

vamos a editar /etc/mkinitcpio.conf:

sudo nano /etc/mkinitcpio.conf


Busca la línea MODULES=() y pon algo asi:

MODULES=(nvidia_drm)


ahora vamos con el hook plymouth en mkinitcpio.conf

sudo nano /etc/mkinitcpio.conf

Busca la línea HOOKS= y vamos a poner plymouth despues de base udev masomenos asi:

HOOKS=(base udev plymouth ...)

agregamos un thea de plymouth:
yay -S plymouth-theme-unrap-git

ahora cargamos el unrap como thema:

sudo plymouth-set-default-theme -R unrap

Editar la entrada normal de systemd-boot

sudo nano /boot/loader/entries/"nombre de tu archivo"

Cambios clave:

En la línea de options añadimos " quiet splash " al final.

Cambiamos loader.conf un poco mejor

vamos a hacer que el timepo de systemboot sea de 0
sudo nano /boot/loader/loader.conf
Y modificamos solamente el:

timeout 0


Guarda y luego:

sudo mkinitcpio -P

Instalar brillo (control de brillo) en directorio temporal

echo "[+] Instalando brillo (control de brillo por terminal)..."
BRILLO_BUILD_DIR="$(mktemp -d)"
(
  cd "${BRILLO_BUILD_DIR}"
  git clone https://gitlab.com/cameronnemo/brillo.git
  cd brillo
  make
  sudo make install
  sudo make install GROUP=video || true
  sudo make install.setgid GROUP=video || true
)
rm -rf "${BRILLO_BUILD_DIR}" || true
