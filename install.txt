    Si está utilizando el systemd Hook, debe ser antes plymouth.

    Por último, regenerar los initramfs. 
    ejemplo:
    Añadir plymouth Al HOOKS Arregle en mkinitcpio.conf.
        HOOKS=(... plymouth ...)

    sudo pacman -S brightnessctl

ahora vamos a configurar el brillo 

    git clone https://gitlab.com/cameronnemo/brillo.git
    cd brillo
    make
    sudo make install
Opcionalmente — para permitir que usuarios normales cambien el brillo (sin necesidad de root) — puedes instalar con permisos de grupo o setgid, así

sudo make install GROUP=video
sudo make install.setgid GROUP=video


instalamos auto-cpufreq
    yay -S auto-cpufreq

activamos el servicio:
    sudo systemctl enable --now auto-cpufreq.service

ahora instalaremos los drivers de intel:
    sudo pacman -S --needed mesa mesa-utils intel-media-driver vulkan-intel libva-intel-driver


ahora instalamos los drivers de nvidia:
    sudo pacman -S --needed nvidia nvidia-utils nvidia-settings nvidia-prime vulkan-icd-loader

    sudo pacman -S --needed lib32-mesa lib32-nvidia-utils lib32-vulkan-intel lib32-vulkan-icd-loader

ahora configuramos el tema:
instalamos dependencias
    sudo pacman -S --needed qt6-svg qt6-virtualkeyboard qt6-multimedia-ffmpeg qt6-5compat qt6-declarative

clonamos el repositorio:
    git clone https://github.com/Keyitdev/sddm-astronaut-theme.git
    cd sddm-astronaut-theme
    sudo mkdir -p /usr/share/sddm/themes
    sudo cp -r src/sddm-astronaut-theme /usr/share/sddm/themes/


ahora le decimos a sddm que tema usar:
sudo mkdir -p /etc/sddm.conf.d

sudo tee /etc/sddm.conf.d/10-theme.conf >/dev/null <<'EOF'
[Theme]
Current=sddm-astronaut-theme
EOF

copiamos las fuentes:
sudo cp -r /usr/share/sddm/themes/sddm-astronaut-theme/Fonts/* /usr/share/fonts/
sudo fc-cache -fv
